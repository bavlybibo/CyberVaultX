# Usage Guide

## Start

```bash
python main.py
```

Create a vault, choose an owner name, and set a strong master password.

## Add Credentials

Use the Vault page to add credentials. Passwords are hidden by default. Copy/reveal actions are logged without exposing the secret.

## Analyze Passwords

Use Password Analyzer to test strength, entropy, patterns, context fit, and local breach-subset status. Warnings are evidence-based and avoid exaggerated severity.

## AI-style Deterministic Security Coach

Open AI-style Deterministic Security Coach after adding credentials. Findings show severity, confidence, evidence, why it matters, and a recommended action.

## Reports

Use Reports to export HTML, JSON, or text. Privacy-safe mode is recommended for demos, GitHub screenshots, and sharing.

## Backup / Recovery

Create encrypted backups with a separate passphrase. Preview restore before importing. Use replace mode only after confirmation.

## System Health

Use System Health to verify runtime, crypto providers, dependency status, storage writeability, and release files.
